Odin Calculator

Uma calculadora simples desenvolvida com HTML, CSS e JavaScript, como parte do The Odin Project, com suporte a mouse e teclado, animações e tratamento de erros.

Funcionalidades

Operações básicas:

Adição (+)

Subtração (-)

Multiplicação (×)

Divisão (÷)

Suporte completo a teclado

Animação visual ao pressionar botões

Botão Backspace para apagar dígitos

Botão AC para limpar tudo

Suporte a números decimais

Resultado limitado a 3 casas decimais

Tratamento de erro para divisão por zero e 
operações inválidas


Controles do Teclado:

Tecla Ação
0–9	Inserir números
.	Decimal
+ - * /	Operadores
Enter ou =	Calcular
Backspace	Apagar último dígito
Esc	Limpar (AC)
